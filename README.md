# Energy Modelling Notes

A collection of notes, tutorials, and resources for energy system modelling, built with [Jupyter Book](https://jupyterbook.org/).

**Website**: [https://YOUR_USERNAME.github.io/energy_modelling_notes/](https://YOUR_USERNAME.github.io/energy_modelling_notes/)

## Topics Covered

- Energy system fundamentals
- Electricity market modelling
- Optimization techniques
- Capacity expansion planning
- Renewable energy integration

## Usage

### Building the book locally

1. Clone this repository
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Build the book:
   ```bash
   jupyter-book build energy_modelling_notes/
   ```
4. View the book:
   - Open `energy_modelling_notes/_build/html/index.html` in your browser

### Deployment
