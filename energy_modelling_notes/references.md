# References and Resources

## Software and Tools

### Modeling Frameworks
- [PyPSA](https://pypsa.org/) - Python for Power System Analysis
- [Linopy](https://linopy.readthedocs.io/) - Linear optimization in Python
- [Pyomo](http://www.pyomo.org/) - Python optimization modeling

### Data Processing
- [Pandas](https://pandas.pydata.org/) - Data analysis library
- [NumPy](https://numpy.org/) - Numerical computing
- [GeoPandas](https://geopandas.org/) - Geospatial data

### Visualization
- [Matplotlib](https://matplotlib.org/) - Static plotting
- [Plotly](https://plotly.com/python/) - Interactive visualization

## Data Sources

### Energy Data
- [ENTSO-E Transparency Platform](https://transparency.entsoe.eu/) - European grid data
- [EIA](https://www.eia.gov/) - US energy statistics
